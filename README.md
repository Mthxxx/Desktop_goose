🪿 Desktop Goose BR

Um projeto inspirado no clássico Desktop Goose, recriado em Python com sprites personalizados, animações e interações divertidas na tela.

✨ Funcionalidades
🪿 Ganso andando livremente pela tela
🖱️ Interação com o mouse
😴 Sistema de descanso/animações
🍃 Evento aleatório de folhas aparecendo na tela
💥 Explosão de folhas quando o ganso alcança o alvo
🎞️ Sprites animados por direção
🪟 Janela fixa com movimentação interna do personagem
📂 Sistema de assets organizado

🚀 Como Executar
1️⃣ Clone o projeto
git clone https://github.com/seuusuario/desktop-goose.git
2️⃣ Entre na pasta
cd desktop-goose
3️⃣ Instale as dependências
pip install -r requirements.txt
4️⃣ Execute o projeto
python script.py
🍃 Sistema de Folhas

A cada intervalo aleatório:

folhas aparecem na tela
o ganso corre até elas
ao chegar:
animação especial
explosão de folhas
retorno ao comportamento normal
